利用hexo搭建个人博客

2021-03-12 | hexo搭建
条评论

个人花了一天多的时间从0开始学习利用hexo框架在github平台上搭建个人博客，网上多数教程只是告诉你怎么做，但一味地跟着步骤走，会让人不知道每一步是干什么的，对于之后的个性化博客建设会产生阻碍。所以我就先从结构上开始讲，解释是什么，再讲怎么做，对小白会很友好。

一个个人博客是怎么搭建的？

一般创建博客要经历这么几个过程：

  1. 本地写好网页界面代码
  2. 写好要发表的文章
  3. 把1和2写好的内容在本地编译成一个网页程序
  4. 把这个程序放在github上

其实抽象后的过程还是很简单的，那么hexo是什么东西呢？它就是一个网页框架，相当于已经把第一步的代码给你写好了，你只需要写好自己的文章，然后放在它的目录下，就可以输入命令完成第3步和第4步。

因此我们在第1步就是下载hexo这个程序框架，可以查看其官网教程，hexo依赖Node.js和git两个软件，因此还要先装上这两个软件，就能直接安装hexo了，只需要在命令行输入命令npm install -g hexo-cli即可

安装了hexo框架，下面我们要生成一个文件夹，用来管理我们的网页，在命令行输入hexo init [folder]就会在当前命令行所在目录下生成一个名为[folder]的文件夹，这个folder可以改成自己的名字（我的就是Blog）。注意，如果打开命令行没有切换目录而直接输入指令，那么就是在C盘的用户目录下生成，可能会费些时间去找这个目录。

生成了文件夹之后，命令行打开文件夹cd Blog，然后就可以直接进行第3步，把网页生成出来，即便还什么都没有加，命令为hexo g或hexo generate，其实前者是后者的简写。

接下来还要运行这个网页，命令为hexo s或hexo server，命令行会出现http://localhost:4000/的内容，把这个地址复制到浏览器打开，就能看到一个网站了（即便里面还没有文章）。
